## Interface de Gestion

Chaque utilisateur dispose d'une fiche affichant :

**Informations**
- Email, téléphone, date inscription
- Dernière activité

**Statistiques**
- Commandes confirmées/annulées
- Taux de réussite

**Actions**
- Modifier, activer/désactiver, supprimer
- Envoi email direct

## Opérations CRUD

### Création

```php
public function store(Request $request)
{
    $validated = $request->validate([
        'name' => 'required|string|max:255',
        'email' => 'required|email|unique:users',
        'role' => 'required|in:moderator,employee',
    ]);
    
    $user = User::create($validated);
    $user->sendWelcomeNotification();
    
    return redirect()->route('users.index');
}
```

### Filtres Disponibles

- Rôle (Moderator/Employee)
- Statut (Actif/Inactif)
- Performance
- Date d'inscription

### Soft Delete

Le système conserve l'historique des utilisateurs supprimés pour traçabilité.

```php
// Suppression logique
$user->delete();

// Restauration
$user->restore();
```
