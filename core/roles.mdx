## Matrice des Permissions

```php
// config/permissions.php
'admin' => ['*'], // Tous les accès
'moderator' => ['products.*', 'orders.*', 'customers.*'],
'employee' => ['products.view', 'orders.*', 'stock.*'],
'media_buyer' => ['campaigns.*', 'analytics.marketing'],
'shipper' => ['deliveries.*', 'sub_shippers.*'],
'sub_shipper' => ['deliveries.assigned', 'packages.scan'],
```

