## Implémentation

```php
// Service DeepSeek
public function chat(string $message, array $context)
{
    return Http::withHeaders([
        'Authorization' => 'Bearer ' . $this->apiKey,
    ])->post('https://api.deepseek.com/v1/chat', [
        'model' => 'deepseek-chat',
        'messages' => [
            ['role' => 'system', 'content' => $this->buildPrompt($context)],
            ['role' => 'user', 'content' => $message]
        ]
    ]);
}
```
